1. 先准备“非默认当前实例 + 一个可观察设置项”
2. 先跑“当前实例主线”校验，确保后面所有页面都围绕同一个实例
3. 依次点按 `Gateway → Models → Chat Channels → System/Settings → Tasks`
4. 在 `System/Settings` 阶段顺手改一个用于恢复验证的设置项
