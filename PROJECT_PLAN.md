## 🔧 核心功能详解

### 1️⃣ 对话框

**功能：**
- 连接 `ws://127.0.0.1:18789`
- 实时对话
- 快捷命令：`/doctor`、`/status`、`/skills`
- 多会话管理
- 消息历史

**技术：**
- WebSocket 连接 Gateway
- Markdown 渲染
- 代码高亮

---

### 2️⃣ Gateway管理

**功能：**
- 状态显示（运行/停止）
- 端口显示
- 运行时间
- 启动/停止/重启按钮
- 性能监控（CPU/内存）

**命令：**
```bash
openclaw gateway status
openclaw gateway start
openclaw gateway stop
openclaw gateway restart
```

---

### 3️⃣ 菜单栏托盘

**功能：**
- 实时状态显示
- 任务自动适配（读HEARTBEAT.md）
- 快速打开控制面板
- 刷新状态

**任务适配逻辑：**
```
1. 读取 ~/.openclaw/workspace/HEARTBEAT.md
2. 解析任务列表
3. 失败 → 提供手动配置
```

---

### 4️⃣ 模型管理

**功能：**
- 当前模型显示
- 快捷切换下拉
- 模型列表
- 添加新模型
- API Key 配置

**命令：**
```bash
openclaw models list
openclaw models set-default <id>
```

**菜单栏快捷切换：**
```
🤖 模型: GPT-5.4 ✓
  ├─ GPT-5.4
  ├─ MiniMax-M2.5
  ├─ Qwen3 (本地)
  └─ 管理模型...
```

---

### 5️⃣ Doctor修复

**四个快捷按钮：**

| 按钮 | 命令 | 功能 |
|------|------|------|
| 🏥 健康检查 | `openclaw doctor` | 基础检查 |
| 🔧 一键修复 | `openclaw doctor --repair` | 自动修复 |
| 🔬 深度检查 | `openclaw doctor --deep` | 严格检查 |
| ⚙️ 配置修复 | `openclaw doctor --fix` | 配置修复 |

**结果展示：**
```
📋 诊断结果：
✅ Gateway: 正常
✅ Config: 正常
⚠️  Telegram: 需重新登录
```

---

### 6️⃣ Channels管理

**功能：**
- 显示已连接Channels
- 连接状态
- 添加新Channel
- 断开连接
- 配置编辑

**支持平台：**
- Telegram
- WhatsApp
- iMessage
- Discord
- Slack

**命令：**
```bash
openclaw status
openclaw channels login --channel telegram
openclaw channels logout --channel telegram
```

## 📝 HEARTBEAT.md 解析

**格式示例：**
```markdown
# 🐾 定期任务

## 核心任务
1. **小红书** - 宠物避坑指南
2. **B站视频** - 按需生成
3. **天气预测** - 监控交易机会
```

**解析规则：**
1. 查找 `## 核心任务` 或 `## 定期任务`
2. 提取列表项 `1. **任务名** - 描述`
3. 显示在托盘菜单

**手动配置：**
```
如果解析失败 → 打开配置界面 → 手动添加任务
```
