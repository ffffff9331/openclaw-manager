# 🦞 Lobster Manager - 龙虾管理助手

一款专为 OpenClaw 用户打造的桌面管理工具，让 AI 助手配置变得简单高效。

## ⭐ 功能特性

- 🎛️ **Gateway 控制** - 一键启动/停止/重启，随时掌控
- ⚡ **快捷指令** - 内置常用命令，支持自定义指令
- 🤖 **模型管理** - 自由添加自定义 AI 模型
- 📱 **多平台集成** - Telegram、Discord、飞书、Slack、WhatsApp、Signal
- 🔧 **系统诊断** - 快速检测并修复问题

## 🛠️ 本地开发

```bash

# 开发模式

# 构建
```
