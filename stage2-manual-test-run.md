## 二、本轮回归范围

### 已纳入
- `GatewayPage`：真实状态读取 + 控制后刷新
- `DoctorPage`：当前实例诊断与状态展示
- `SettingsPage`：安装 / 部署引导 + 备份最小闭环 + 本地 / 远端边界提示
- `ModelsPage`：模型管理增强
- `TasksPage`：cron 第一版维护闭环
- `AppLogsPage`：当前实例日志查看
- `SkillsPage`：技能列表 / 启用禁用 / 简单编辑入口

### 暂不纳入
- 远端实例自动安装
- 备份还原后的完整业务一致性验证
- 一个 provider 下多模型的复杂拖拽交互体验打磨
- cron 高级编排项：`announce`、`delivery`、`account`、`failure-alert`、`session-key`、`light-context`
